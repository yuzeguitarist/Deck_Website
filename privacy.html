Privacy Policy

We believe privacy is a fundamental human right. Deck was designed with privacy at its core.

Last updated: December 2025

Data Storage

Deck is a fully local application. All clipboard data is stored on your Mac and never uploaded to any server.

  • Clipboard history is stored in a local SQLite database
  • Images and files are stored as blobs within the app sandbox
  • Semantic search vectors are computed and stored locally
  • All settings and preferences are saved in UserDefaults

Data Encryption

Sensitive data is protected using industry-standard encryption algorithms:

  • AES-256-GCM symmetric encryption
  • Encryption keys are stored in the macOS Keychain
  • Support for on-demand encryption of specific clipboard entries
  • LAN sharing uses end-to-end encryption

Biometric Security

Deck supports native macOS biometric protection:

  • Optional Touch ID / Face ID unlock
  • Sensitive content access requires re-authentication
  • Biometric data is handled by the system security module — Deck cannot access it

Sensitive Content Protection

Deck automatically detects and skips sensitive content:

  • Recognizes password manager clipboard types (ConcealedType)
  • Detects window titles containing sensitive keywords
  • Support for manually marking apps as "ignored"
  • Configurable smart rules for automatic filtering

Network Access

Deck's network access is extremely limited:

  • LAN Sharing: P2P connections only within local network
  • Link Previews: Fetches Open Graph metadata from URLs (can be disabled)
  • No telemetry, analytics, or crash report uploads
  • No ad SDKs or third-party tracking

Data Deletion

You can completely delete your data at any time:

  • Delete any individual clipboard entry
  • Clear all history
  • Uninstalling the app removes all local data
  • Export feature lets you backup data before deletion

Open Source Transparency

Deck is open source software. You can:

  • Audit the source code to verify our privacy claims
  • Understand how data is processed and stored
  • Submit security vulnerability reports
  • Contribute to improving privacy-related features

Source code is hosted on GitHub under the MIT license.

For any privacy-related questions, please contact us via GitHub Issues.
