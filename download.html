Installation Guide

01

Download DMG

Click the Universal Build above to get the latest disk image.

02

Install App

Open the DMG and drag Deck to your Applications folder.

03

Grant Permissions

Grant Accessibility permissions in System Settings on first launch (and Input Monitoring if prompted).

04

Start Using

Press ⌘ + P to open the panel.
