Comprehensive Capabilities

All Features

Built natively with Swift, deeply optimized for macOS. Every detail has been carefully crafted to respect your privacy and boost your workflow.

Intelligent Search

Semantic Search

Local vector database powered by NLEmbedding. Search with natural language - no need to remember exact keywords, just describe what you're looking for.

Smart Content Recognition

Automatically identifies text, rich text, code, URLs, color codes, images, files and more. Provides tailored previews and actions for each type.

OCR Text Extraction

Automatically extract text from images, making screenshot content searchable and copyable.

Clipboard Management

Multi-Format Support

Perfect support for plain text, rich text (RTF/RTFD), images, file paths, URLs and more. Preserves original formatting when pasting.

Smart Clean Paste

Automatically detects target apps and intelligently switches to plain text paste in Word/Excel, preventing format conflicts.

URL Tracking Removal

Automatically removes tracking parameters (utm_*, fbclid, etc.) from URLs, protecting your privacy.

Smart Line Break Removal

Automatically removes excess line breaks from text copied from PDFs, browsers and more for cleaner output.

Rich Previews

Links, PDFs, images with inline preview overlay. See what you copied at a glance without opening files.

Large Payload Handling

Handles large content smoothly without lag. Copy entire documents, high-res images, or lengthy code snippets effortlessly.

Groups & Tags

Custom Groups

Create color-coded group tags to categorize your clipboard content. Supports drag-and-drop sorting.

Smart Rules

Automatically group or ignore content based on content type, source app, keywords and more.

Paste Queue

Select multiple items to add to a queue and paste them in sequence, improving batch processing efficiency.

Privacy & Security

Local First

All data stored in local SQLite database. Nothing uploaded to the cloud. Your data, your control.

Biometric Protection

Support for Touch ID / Face ID unlock. Sensitive content requires verification to access.

Sensitive Content Detection

Automatically detects password managers, login pages and other sensitive windows, skipping recording to protect privacy.

AES-256 Encryption

Uses CryptoKit for AES-GCM encryption of sensitive data. Keys stored securely in Keychain.

Auto-Cleanup

Configure automatic deletion of old clipboard items. Keep your history clean and storage optimized.

App Exclusion List

Exclude specific apps from clipboard monitoring. Perfect for banking apps or confidential software.

Local Network Sharing

P2P Direct Connect

Peer-to-peer connection via MultipeerConnectivity. No server required, no internet needed.

TOTP Verification

Secure mode uses one-time verification codes to establish encrypted connections, preventing unauthorized access.

Group Sharing

Share entire groups to other devices with one click, including all items and metadata.

System Integration

Global Shortcuts

Customize keyboard shortcuts to quickly open the panel, search, switch groups and more.

Shortcuts Integration

Support for macOS Shortcuts, access clipboard history in your automation workflows.

Script Plugins

Support for AppleScript and JavaScript plugins for custom content processing logic.

Menu Bar Resident

Lightweight menu bar icon shows clipboard status in real-time. One-click pause/resume recording.

Vim Navigation

Optional Vim-style keyboard navigation (j/k/x/y) for power users who love keyboard-first workflows.

Dark/Light Mode

Follows system theme automatically. Native macOS design that feels right at home on your Mac.
