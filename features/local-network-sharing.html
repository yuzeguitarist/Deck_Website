LAN sharing FAQ

Is this the same as full cloud clipboard sync?

No. This feature is designed around local network sharing between nearby devices, which is different from continuously syncing clipboard history through a public cloud service.

How is the connection protected?

Deck uses encrypted transport for local sharing and includes device verification so users can control which devices participate in a session.

Who benefits from this feature?

People who use more than one Mac in the same environment, teams working on a shared LAN, and privacy-conscious users who want controlled sharing without a default internet relay.
