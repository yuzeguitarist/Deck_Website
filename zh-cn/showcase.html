Introducing Deck Script Plugin

你负责开口，Deck 负责把它变成可用插件。

Deck Script Plugin 让你直接用自然语言描述想要的脚本，然后把这个想法变成真正能跑的插件。现在已经不需要你懂 JavaScript 了。

它可以访问网络，也能一键自动化处理复杂又琐碎的任务，像剪贴板归类这种需求都能轻松搞定。这个演示里，AI 先写出一个脚本插件，再立刻用它去处理一段文本，速度非常快，而且处理结果很聪明。

Deck v1.3.2
认识 Deck AI Memory

记忆，会被每一次对话慢慢塑形。

Deck AI Memory 会自动捕捉你在交流里透露出的记忆信息，并整理成一份不断生长的数字画像。你和它聊得越多，它就越懂你的习惯、偏好和当下语境。

这些内容会加密保存在本地文件里，只有这个 AI 自己能访问，所以不用担心安全和隐私问题。
