功能全览

所有 功能

Swift 原生构建，为 macOS 深度优化。现已内置 AI 助手，支持多模型接入与审批式工具调用——本地优先隐私不打折扣。

剪贴板核心

剪贴板历史

统一记录文本、富文本、图片、文件、颜色和链接等内容。

丰富预览

链接、PDF、图片和长文本都可直接预览后再粘贴。

大内容处理

面对大段文本、文档和图片仍保持流畅搜索与调用。

智能纯净粘贴

在易出格式问题的应用里自动切换为纯文本粘贴。

检索与找回

快速搜索

支持关键词、正则、类型筛选和语义匹配，快速找回历史内容。

智能内容识别

自动识别内容类型，并提供对应的预览和操作。

OCR 文字提取

自动提取图片文字，让截图内容也能被搜索。

链接净化

自动移除 URL 中的 utm_*、fbclid 等追踪参数。

智能换行移除

自动修正常见的 PDF/网页断行问题，粘贴更干净。

即时计算

可直接对复制内容中的数学表达式进行快速计算。

AI、规则与队列

标签与智能分类

用颜色标签和分组管理历史，结构清晰，查找更快。

智能规则

按应用、内容类型、关键词或模式自动分组、转换或忽略。

Deck AI 助手

可用自然语言直接询问历史内容，完成检索、总结和文本改写等任务。

多模型提供商

可按需求切换 ChatGPT、OpenAI API、Anthropic 或本地 Ollama。

AI 操作审批

写入、删除、脚本插件改动等敏感操作都需要你确认后才会执行。

JavaScript 脚本插件

通过 JavaScript 或 AppleScript 插件扩展内容处理逻辑。

上下文感知排序

根据当前应用优先展示更相关的剪贴内容。

队列模式

支持多条记录按顺序粘贴，完整适配键盘优先流程。

隐私与安全

本地优先存储

剪贴板数据默认只存本机，由你完全掌控。

Touch ID 保护

访问敏感内容前可要求 Touch ID 二次验证。

敏感内容检测

自动跳过密码管理器及敏感窗口中的复制内容。

AES-GCM 加密

敏感数据采用现代加密方案并安全管理密钥。

应用排除列表

可将特定应用排除在监控外，保护隐私场景。

自动清理

按保留策略自动清理旧记录，减少历史负担。

剪贴板隐写

支持图片/零宽文本隐写与还原，满足进阶场景。

局域网共享

局域网 P2P 共享

同一局域网设备间直接共享剪贴板内容。

TOTP 验证

共享配对时可使用一次性验证码提升安全性。

分组共享

支持整组内容连同元数据一次性共享。

系统与扩展

全局快捷键

自定义呼出、搜索、分组切换等核心快捷操作。

Vim 导航

提供可选 j/k/x/y 键位，适配键盘优先用户。

macOS 快捷指令

可在快捷指令里直接读取和处理 Deck 历史。

菜单栏常驻

在菜单栏快速查看状态并暂停/恢复记录。

迁移助手

支持从 Maccy、Paste、PasteNow、Flycut、Pastebar 导入历史。

自动更新

默认每日检查更新，也可在设置里手动检查。

CLI Bridge

可通过命令行与本地脚本、外部工具进行联动。

深色/浅色模式

自动跟随系统外观，保持原生 macOS 体验。
