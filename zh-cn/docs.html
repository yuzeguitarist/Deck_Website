学习 Deck

官方指南——从快速上手到进阶工作流，涵盖 AI 助手配置、隐私保护、局域网共享与自动化扩展。

快速入门

01

安装 Deck

下载 DMG 文件并将 Deck 拖入应用程序文件夹。

02

授予权限

在系统设置 > 隐私与安全中允许辅助功能权限（如有提示，也请开启输入监控）。

03

打开 Deck

按 ⌘ + P 从任何地方打开 Deck 面板。

04

开始复制

你复制的所有内容都会自动保存。随时搜索和粘贴。

键盘快捷键

一览所有键盘快捷键，让操作快人一步。

  • ⌘P — 打开 Deck 面板
  • ⌥Q — 切换队列模式
  • ⌘⇧V — 队列模式下顺序粘贴
  • ↑ / ↓ — 导航项目
  • Enter — 粘贴选中项
  • Shift + Enter — 粘贴为纯文本
  • ⌘ + 数字 — 快速粘贴 (1-9)
  • Delete — 删除选中项
  • Esc — 关闭面板

检索与找回

结合关键词、正则、类型筛选与语义匹配，快速找回内容。

  • 关键词检索：适合精确文字
  • 正则检索：适合验证码、编号等模式内容
  • 按类型筛选：文本、图片、文件、链接、颜色
  • 加来源应用条件：如“来自 Safari / VS Code”
  • 只记得大意时使用语义匹配

设置与工作流

根据日常习惯精细调整 Deck。

  • 设置全局呼出快捷键（默认 ⌘P）
  • 配置历史保留策略与自动清理
  • 调整队列模式和粘贴行为
  • 设置应用排除列表与敏感窗口规则
  • 开启上下文感知排序和智能规则
  • 配置快速粘贴修饰键与菜单栏行为

Deck AI 助手

配置 AI 提供商、安全使用 AI 对话，通过审批机制控制敏感操作。

  • 默认快捷键 Option + Shift + Space 打开 AI 聊天窗口
  • 可在设置里选择 ChatGPT、OpenAI API、Anthropic 或本地 Ollama
  • 可附带剪贴板内容，让 AI 做总结、改写、提取重点
  • 支持用内置 AI 工具检索历史并运行脚本转换
  • 写入/删除/插件改动等操作默认需要审批确认

隐私与安全

以本地优先能力守护你的敏感数据。

  • 历史数据默认仅存于本机
  • 为敏感内容启用 Touch ID 保护
  • 为受保护条目启用 AES-GCM 加密
  • 自动跳过密码管理器和敏感窗口
  • 默认不上传分析数据；仅在你主动开启“上传分析数据”后按说明上传；无第三方追踪 SDK

局域网共享

在本地网络中跨设备共享剪贴板内容。

  • 在设置中开启局域网 P2P 共享
  • 设备需处于同一局域网
  • 通过 TOTP 验证完成安全配对
  • 可共享单条内容或整个分组
  • 无需云账户，传输为端到端加密

自动化与扩展

通过快捷指令、脚本插件和 CLI Bridge 拓展 Deck 的能力边界。

  • 通过 macOS 快捷指令读取或搜索历史
  • 用 JavaScript/AppleScript 插件构建处理流程
  • 通过 CLI Bridge 触发本地自动化
  • 结合迁移/导出做备份流转
  • 与本地工具链及 Orbit 工作流联动

常见问题

Deck 支持 Intel Mac 吗？

是的！Deck 是通用二进制文件，可在 Apple Silicon 和 Intel Mac 上原生运行。

我的剪贴板数据会上传到云端吗？

默认不会。剪贴板数据默认存储在你的 Mac 本地；仅当你主动开启 iCloud 同步或“上传分析数据”等功能时，才会有对应上行。

Deck AI 能访问什么内容？

Deck AI 基于你的本地会话和剪贴板上下文工作。写入、删除、插件修改这类敏感操作都要你手动确认后才会执行。

我可以在多台 Mac 之间同步剪贴板吗？

可以，使用 P2P 网络共享功能。两台 Mac 需要在同一本地网络上。

如何排除敏感应用？

前往偏好设置 > 隐私，将应用添加到排除列表。Deck 将跳过记录这些应用的内容。

达到历史记录上限会怎样？

达到上限时，最旧的项目会自动删除。置顶的项目永远不会被删除。
