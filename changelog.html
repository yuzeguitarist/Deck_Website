1.1.1

January 1, 2026Latest

Safer!

Highlights

  • Bank card number recognition with multi-strategy validation
  • Custom template libraries with trigger keywords
  • Fixed crashes and authentication dialog issues
New Features
  • Bank card number recognition with multi-strategy validation (length matching, BIN prefix rules, Luhn algorithm)
  • Custom template libraries with per-library trigger keywords for Cursor Assistant
Bug Fixes
  • Fixed crash when clicking refresh button next to connected devices on Network page
  • Fixed fingerprint authentication dialog flickering and repeatedly appearing after cancellation in Secure Mode
1

1.1.0 (fix)

January 1, 2026

Bug Fix!

Highlights

  • All database access now runs asynchronously
  • Fixed main-thread blocking causing battery drain
  • Search, FTS, stats and export paths optimized
Bug Fixes
  • All DB access now runs asynchronously on dbQueue with await, eliminating main-thread blocking
  • Search, FTS, vector queries, and migration paths now use async DB calls
  • Stats and export now read the DB asynchronously, UI thread no longer executes SQL directly
  • Async closures now explicitly capture self for clearer concurrency semantics

Huge Update!

Highlights

1.0.9

December 28, 2025

More Composed!

Highlights

  • Semantic search engine upgrade with faster responses
  • Memory optimization: 300MB → 50MB
  • Enhanced Chinese search experience
Improvements
  • Semantic search engine upgrade: Apple Sentence Embedding + sqlite-vec static integration
  • Smarter hybrid mode: Text and semantic results merged for higher relevance
  • Enhanced Chinese search: FTS5 trigram tokenization for more accurate CJK retrieval
  • OCR performance optimization: Downsampling for large images to reduce lag
  • Sync and database stability: CloudKit batch submission, automatic backup/integrity checks
  • Memory optimization: Reduced usage from 300MB to 50MB

1.0.0

December 6, 2025

Deck Launches!

Highlights

  • Semantic Search with NLEmbedding
  • Privacy First with local storage and Touch ID
  • LAN P2P Sharing without servers
New Features
  • Instant clipboard history - text, images, links, files
  • Fuzzy search - find any clip in milliseconds

  • Smart rules - customizable automation
  • Plugin system - extend workflows with scripts
  • Rich previews - links, PDFs, images
  • P2P LAN transfer - no cloud needed
  • Keyboard-first - Cmd+P, Vim navigation (j/k/x/y)
  • Native SwiftUI + AppKit experience
