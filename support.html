Common Issues

Deck doesn't open or show in menu bar

Make sure you've granted Accessibility permissions in System Preferences > Privacy & Security > Accessibility. Restart Deck after granting permissions.

Clipboard history is not being recorded

Check if the app you're copying from is in the exclusion list. Also verify that recording isn't paused (click the menu bar icon to check).

Search results don't match expectations

Semantic search works best with descriptive queries. Try different phrasings or use exact match with quotes for specific text.

P2P sharing not finding other devices

Ensure both devices are on the same Wi-Fi network with Bluetooth enabled. Check that firewall settings allow local network connections.
