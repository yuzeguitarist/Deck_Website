Common Issues

Deck doesn't open or show in menu bar

Make sure you've granted Accessibility permissions in System Settings > Privacy & Security > Accessibility. If prompted, also enable Input Monitoring. Restart Deck after granting permissions.

Clipboard history is not being recorded

Check whether recording is paused and whether the source app is in the exclusion list. Deck also skips password managers and sensitive windows by design.

Search results don't match expectations

Try keyword/regex/type filters first, then semantic matching. Adding source app or time range usually improves recall.

LAN sharing cannot find other devices

Ensure both devices have LAN sharing enabled, are on the same local network, and complete TOTP verification if secure pairing is enabled.
