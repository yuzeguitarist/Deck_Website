Learn Deck

Official guides — from quick start to advanced workflows, AI assistant setup, privacy controls, LAN sharing, and automation.

Quick Start

01

Install Deck

Download the DMG file and drag Deck to your Applications folder.

02

Grant Permissions

Allow Accessibility permissions in System Settings > Privacy & Security (and Input Monitoring if prompted).

03

Open Deck

Press ⌘ + P to open the Deck panel from anywhere.

04

Start Copying

Everything you copy will be saved automatically. Search and paste anytime.

Keyboard Shortcuts

All keyboard shortcuts for a lightning-fast workflow.

  • ⌘P — Open Deck panel
  • ⌥Q — Toggle Queue Mode
  • ⌘⇧V — Paste next item in Queue Mode
  • ↑ / ↓ — Navigate items
  • Enter — Paste selected item
  • Shift + Enter — Paste as plain text
  • ⌘ + Number — Quick paste (1-9)
  • Delete — Delete selected item
  • Esc — Close panel

Search & Recall

Find any copied content with keyword, regex, type filters, or semantic matching.

  • Use keyword search for exact text
  • Use regex for structured patterns like codes or IDs
  • Filter by type: text, image, file, link, color
  • Add source app context like "from Safari" or "from VS Code"
  • Use semantic matching when you only remember rough meaning

Settings & Workflow

Fine-tune Deck to match your daily workflow.

  • Set global panel shortcut (default ⌘P)
  • Configure history retention and auto-cleanup
  • Adjust queue mode and paste behavior
  • Set app exclusion list and sensitive-window rules
  • Enable context-aware ordering and smart rules
  • Configure quick paste modifier and menu bar behavior

Deck AI Assistant

Configure providers, chat with AI safely, and automate clipboard tasks with approval controls.

  • Open AI chat with Option + Shift + Space (default shortcut)
  • Choose ChatGPT, OpenAI API, Anthropic, or local Ollama in settings
  • Attach clipboard content and let AI summarize, rewrite, or extract key info
  • Use built-in AI tools to search history and run script transforms
  • Write/delete/plugin operations require explicit approval before execution

Privacy & Security

Keep sensitive clipboard data safe with local-first controls.

  • All history is stored locally on your Mac
  • Enable Touch ID protection for sensitive clips
  • Enable AES-GCM encryption for protected entries
  • Auto-detect and skip password manager windows
  • No third-party tracking SDKs; diagnostics upload is off by default and requires opt-in

LAN Sharing

Share clipboard items across devices on your local network.

  • Enable LAN P2P sharing in settings
  • Devices must be on the same local network
  • Use TOTP verification for secure pairing
  • Share single items or whole groups
  • No cloud account required, transfer is end-to-end encrypted

Automation & Extensions

Extend Deck with shortcuts, scripts, and CLI bridge.

  • Trigger local workflows through CLI Bridge
  • Combine with migration/export for backup workflows
  • Integrate with your local tools and Orbit workflow

FAQ

Does Deck work on Intel Macs?

Yes! Deck is a Universal Binary that runs natively on both Apple Silicon and Intel Macs.

Is my clipboard data sent to the cloud?

By default, clipboard data stays on your Mac. Data is uploaded only if you explicitly enable features such as iCloud sync or diagnostics upload.

What can Deck AI access?

Deck AI works with your local conversation and clipboard context. Sensitive actions such as writing, deleting, or plugin modification always require your approval.

Can I sync clipboard between Macs?

Yes, using the P2P network sharing feature. Both Macs need to be on the same local network.

How do I exclude sensitive apps?

Go to Preferences > Privacy and add apps to the exclusion list. Deck will skip recording from those apps.

What happens if I reach the history limit?

Oldest items are automatically removed when the limit is reached. Pinned items are never deleted.
