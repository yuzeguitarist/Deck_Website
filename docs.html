Learn Deck

Everything you need to get the most out of Deck. From quick start to advanced features.

Quick Start

01

Install Deck

Download the DMG file and drag Deck to your Applications folder.

02

Grant Permissions

Allow Accessibility permissions in System Preferences > Privacy & Security.

03

Open Deck

Press ⌘ + Shift + V to open the Deck panel from anywhere.

04

Start Copying

Everything you copy will be saved automatically. Search and paste anytime.

Keyboard Shortcuts

Master all keyboard shortcuts for lightning-fast workflow.

  • ⌘ + Shift + V — Open Deck panel
  • ⌘ + F — Focus search bar
  • ↑ / ↓ — Navigate items
  • Enter — Paste selected item
  • ⌘ + Enter — Paste as plain text
  • ⌘ + Delete — Delete selected item
  • Esc — Close panel

Semantic Search

Search your clipboard history using natural language.

  • Type naturally: "the email I copied yesterday"
  • Search by content type: "images", "links", "code"
  • Search by app: "from Safari", "from VS Code"
  • Use quotes for exact match: "exact phrase"
  • Combine filters: "code from VS Code last week"

FAQ

Does Deck work on Intel Macs?

Yes! Deck is a Universal Binary that runs natively on both Apple Silicon and Intel Macs.

Is my clipboard data sent to the cloud?

No. All data is stored locally on your Mac. Nothing is ever uploaded to any server.

Can I sync clipboard between Macs?

Yes, using the P2P network sharing feature. Both Macs need to be on the same local network.

How do I exclude sensitive apps?

Go to Preferences > Privacy and add apps to the exclusion list. Deck will skip recording from those apps.

What happens if I reach the history limit?

Oldest items are automatically removed when the limit is reached. Pinned items are never deleted.
